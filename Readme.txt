Для проверки задания необходимо:
1. Скачать и установить NodeJs.
2. Установить Gulp.
3. Установить зависимости проекта (npm i)
4. Запуск осуществляется командой gulp, из директории с проектом. 

В проекте осуществлена возможность загрузки изображений методом Drag-n-Drop,
для этого необходимо перетащить изображение в область формы, она выделится сплошной границей.

Так же изображение можно добавить введя ссылку на изображение в поле input, и нажав кнопку "Загрузить".
Осуществлена возможность загрузки изображени локально с компьютера. Для этого необходимо нажать "Выбрать изображения".

Для удаления картинки необходимо кликнуть на картинку, она выделится синим цветом, затем необходимо нажать кнопку "Удалить"
